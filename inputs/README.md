# Inputs

This folder contains images used throughout the repo. 'Actual' images are images that are true screenshots taken from the video. 'Recorded' images are images that are photographs of an environment that includes a projector image on a wall.

Actual images were taken from this [YouTube video](https://www.youtube.com/watch?app=desktop&v=yaaE4bCf1UU) and recorded images were taken from this [YouTube video](https://www.youtube.com/watch?app=desktop&v=q3jT8J6VjuY)
